# REGRESI LINEAR
Silakan install beberapa library python berikut ini
1. sklearn
2. matplotlib
3. pandas
4. numpy
5. seaborn

### Instalasi sklearn
```sh
pip install sklearn
```
### Instalasi matplotlib
```sh
pip install matplotlib
```

### Instalasi pandas
```sh
pip install pandas
```

### Instalasi numpy
```sh 
pip install numpy
```

### Instalasi seaborn
```sh
pip install seaborn
```
